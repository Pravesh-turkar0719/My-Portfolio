
# 🌸 Sakura Portfolio | Pravesh Turkar

A modern, responsive, and aesthetically pleasing portfolio built with **React.js**. This project features a unique "Sakura petals on Sky Blue" theme, utilizing smooth animations and a clean UI to showcase my journey as a Frontend Developer and B.Tech student.

🚀 **[Visit My Portfolio](https://my-portfolio-psi-nine-43.vercel.app/)** ---

## ✨ Features

* **Custom Sakura Animation:** Elegant falling petal effect created using pure CSS keyframes.
* **Dynamic Scroll Spy:** Navigation links that automatically highlight based on the current section.
* **Responsive Design:** Fully optimized for all screen sizes (Mobile, Tablet, Desktop).
* **Interactive Skill Bars:** Animated progress bars that trigger when scrolled into view using the `Intersection Observer API`.
* **Optimized Performance:** Fast load times and smooth scrolling for a premium user experience.
* **Data-Driven:** Content is managed centrally via a mock data file for easy updates.

---

## 🛠️ Tech Stack

* **Frontend:** React.js, JavaScript (ES6+)
* **Styling:** CSS3 (Variables & Flex/Grid)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Routing:** React Router
* **Deployment:** Vercel
* **Version Control:** Git & GitHub

⚙️ Local Setup  
To run this project locally, follow these steps:

Clone the repository:

```bash
git clone https://github.com/Pravesh-turkar0719/my-portfolio.git
```

Navigate to the project folder:

```bash
cd my-portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

The site will open at http://localhost:3000.
